# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-07

### Added
- Initial CPU-first restoration + detection recovery MVP.
- Data corruption pipeline for rain, fog, and low-light synthesis.
- U-Net style restoration autoencoder with composite loss.
- CLI tools for training, evaluation, benchmarking, and realtime demo.
- Beginner-focused documentation and test suite.
