# Security Policy

## Supported Versions

This is a research MVP. Security patches are best-effort on the latest default branch.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Instead:
- Share a private report with clear reproduction steps.
- Include affected files, inputs, and observed impact.
- Include version/commit and environment details.

We will acknowledge receipt and provide a remediation timeline when reproducible.

## Scope Notes

This repository focuses on ML experimentation and local workflows.  
Production-grade hardening (authn/authz, secure deployment, secret management) is out of current scope.
